# SundialKitMessagable

**Part of [SundialKit](https://github.com/brightdigit/SundialKit) v2.0.0**

Dictionary-based messaging plugin for SundialKit maintaining v1.0.0 Messagable protocol and MessageDecoder compatibility.

## About

SundialKitMessagable is a compatibility plugin for SundialKit, providing the v1.0.0 Messagable protocol and MessageDecoder for dictionary-based WatchConnectivity messaging. This package maintains backward compatibility with SundialKit v1.0.0 APIs and is part of the SundialKit v2.0.0 monorepo architecture.

## Parent Project

This package is maintained as a git-subrepo within the main SundialKit repository during v2.0.0 development.

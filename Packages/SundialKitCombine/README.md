# SundialKitCombine

**Part of [SundialKit](https://github.com/brightdigit/SundialKit) v2.0.0**

Combine-based compatibility plugin for SundialKit providing v1.0.0 API with NetworkObserver and ConnectivityObserver.

## About

SundialKitCombine is a Layer 2 observation plugin for SundialKit, providing @MainActor-based observers with Combine publishers for SwiftUI integration. This package maintains backward compatibility with SundialKit v1.0.0 APIs and is part of the SundialKit v2.0.0 monorepo architecture.

## Parent Project

This package is maintained as a git-subrepo within the main SundialKit repository during v2.0.0 development.
